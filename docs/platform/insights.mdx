Insights turns indexed code data into dashboards. Use it to answer questions like:

- How many deprecated imports are left?
- Which repositories still need a migration?
- Which packages and versions are installed?
- Is adoption improving over time?

[Open Insights](https://app.codemod.com/insights) in the app to start building dashboards.

## How Insights works

An Insights dashboard is made of widgets. Each widget runs one or more queries against selected repositories, previews the result, and renders it as a number, chart, table, or note.

<Steps>
  <Step title="Create a dashboard">
    Start with one migration, code health area, or dependency question.
  </Step>
  <Step title="Add widgets">
    Choose the visualization that matches the question: timeseries, single number, table, or note.
  </Step>
  <Step title="Choose data sources">
    Query JSSG codemods, ast-grep rules, or dependency data.
  </Step>
  <Step title="Group, filter, and combine results">
    Break results down by repository, directory, owner, or custom metric cardinality, then use formulas when a widget needs more than one query line.
  </Step>
</Steps>
