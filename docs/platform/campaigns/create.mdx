Create a campaign when you are ready to apply a codemod to one or more repositories.

## Choose a codemod

Start from a registry codemod that matches the migration. For example, use `debarrel` to clean up barrel imports or an i18n codemod to roll out localization changes.

## Select target repositories

Pick the repositories you want Codemod to update. Start with one repository when you want a small first run, then expand to multiple repositories after the generated PR looks right.
